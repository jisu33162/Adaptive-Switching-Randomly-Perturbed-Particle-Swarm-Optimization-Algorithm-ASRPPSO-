# Adaptive-Switching-Randomly-Perturbed-Particle-Swarm-Optimization-Algorithm-ASRPPSO-
Source Code of the ASRPPSO Algorithm

This repository provides the source code of the ASRPPSO algorithm proposed in the following paper:

[A New Particle Swarm Optimization Algorithm for Outlier Detection: Industrial Data Clustering in Wire Arc Additive Manufacturing]
[[Paper Link / DOI](https://ieeexplore.ieee.org/abstract/document/10003177)]

Requirements

The code was developed and tested in MATLAB. No additional toolbox is required unless otherwise specified in the source file.

Please note that this repository only provides the data-independent implementation of the ASRPPSO algorithm. The WAAM data used in the associated study are not publicly released due to confidentiality and intellectual property restrictions.
